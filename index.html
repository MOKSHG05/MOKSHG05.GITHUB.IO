TPJ Capstone Project

This project is an industrial-level system to store and retrieve pallets automatically at designated locations with Node-RED interface and stepper motor control. It includes three phases: Design, Engineering, and Manufacturing.

  • Design
  • Engineering
  • Manufacturing

Design Phase

Includes CAD models, layout drawings, and initial system architecture.

Engineering Phase

Manufacturing Phase

Assembly, wiring, fabrication photos, and final testing with real hardware.
